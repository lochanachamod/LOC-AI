# Contributing to LOC-AI

First off, thank you for considering contributing to LOC-AI! It's people like you that make LOC-AI such a great tool for developers.

## How Can I Contribute?

### Reporting Bugs
This section guides you through submitting a bug report for LOC-AI.
- Use the GitHub Issues tab to report any bugs.
- Clearly describe the issue including steps to reproduce.
- Include your OS and LOC-AI version.

### Suggesting Enhancements
- Use the GitHub Issues tab to suggest new features or enhancements.
- Describe how the feature would work and why it would be beneficial.

### Pull Requests
1. Fork the Project
2. Create your Feature Branch (`git checkout -b feature/AmazingFeature`)
3. Commit your Changes (`git commit -m 'Add some AmazingFeature'`)
4. Push to the Branch (`git push origin feature/AmazingFeature`)
5. Open a Pull Request
